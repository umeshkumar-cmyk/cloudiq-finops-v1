## 🔧 Technical Architecture

### Layered Architecture
```
┌─────────────────────────────────────┐
│      UI Layer (Streamlit)           │
│  - 5-tab dashboard                  │
│  - Sidebar configuration             │
│  - Interactive components           │
├─────────────────────────────────────┤
│    Analytics Layer (Modules)         │
│  - DataEngine (load, clean, agg)    │
│  - AnalyticsEngine (compute)        │
│  - Visualizer (charts)              │
├─────────────────────────────────────┤
│      Data Layer (CSV + Memory)       │
│  - CSV file loading                  │
│  - In-memory DataFrames             │
│  - Future: Cloud APIs               │
└─────────────────────────────────────┘
```

### Data Pipeline
```
CSV Input
    ↓
Load Data (pandas.read_csv)
    ↓
Validate Schema
    ↓
Normalize Data (types, dates, values)
    ↓
Filter by Criteria (date range, etc.)
    ↓
Aggregate (daily → monthly)
    ↓
Analyze (anomalies, forecast, etc.)
    ↓
Visualize (Plotly charts)
    ↓
Present (Streamlit UI)
```

### Algorithm Implementations

#### Anomaly Detection
```python
For each service:
  1. Calculate baseline (mean of historical data)
  2. Get current value (last data point)
  3. Calculate % change = (current - baseline) / baseline * 100
  4. Flag if change > threshold_pct
  5. Classify severity based on magnitude
Return sorted by severity
```

#### Forecasting
```python
1. Aggregate data to monthly level
2. Create X values (month index: 0, 1, 2, ...)
3. Create y values (spend amounts)
4. Fit polynomial of degree N (default: 2)
5. Calculate coefficients using np.polyfit
6. Generate future month indices
7. Predict values using fitted polynomial
8. Return historical + forecast
```

#### Optimization Recommendations
```python
For each service/provider combination:
  IF efficiency_score < 50:
    recommend = "Rightsizing"
  ELIF usage > 50 AND efficiency > 75:
    recommend = "Reserved Instances"
  ELIF usage < 10 AND spend > 100:
    recommend = "Resource Cleanup"
  
  Calculate potential_savings based on type
  Assign priority based on savings impact
```
